Create a CSV File

Overview

CSV (Comma-Separated Values) files are a simple way to organize data in plain text. In Edwin, they make it easy to upload multiple entries - like student email addresses or school locations - all at once.

Each line in a CSV file represents one item, and because this format works with tools like Microsoft Excel, Google Sheets, and most text editors, it’s quick to create, open, and share.

Using CSV files helps keep your data structured and accurate during uploads.

Note

The format for both student and location uploads is the same:

  • Do not include a header row.
  • The first row cannot be blank.
  • Enter one item per row in a single column. 
  • Save CSV files using UTF-8 encoding to ensure all characters, including non-Latin ones, display correctly.
